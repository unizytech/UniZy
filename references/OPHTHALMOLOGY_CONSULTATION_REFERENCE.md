## Layout Notes

### Page Structure
The form is organized across 4 pages:
- **Page 1**: Patient details, history, refraction, K reading, cover tests
- **Page 2**: Visual tests, color vision, PBCT charts, diplopia charting
- **Page 3**: Dry eye assessment, slit lamp examination, IOP, gonioscopy
- **Page 4**: Fundus examination, diurnal IOP, HVF, diagnosis, treatment plan, notes

### Design Patterns
1. **Bilateral Data**: Most sections have OD (right eye) / OS (left eye) columns
2. **Tables**: Frequently used for structured bilateral data comparison
3. **Text Fields**: Mix of single-line fields and multi-line text areas
4. **Grid Layouts**: PBCT uses 3x3 grids for prism cover test results
5. **Radio Buttons**: Used for binary choices (e.g., Dilated/undilated fundus)
6. **Headers**: Section headers in bold uppercase for clear visual hierarchy
7. **Subsections**: Nested organization with indentation for related fields

### Data Types by Section
- **Numeric**: Vision measurements (6/60), IOP readings (mm Hg), CDR ratios
- **Categorical**: Test results (Normal, Clear, G1-2, NS1-2, PERRLA)
- **Free Text**: History, complaints, doctor notes
- **Time Series**: Diurnal IOP with multiple time-stamped readings
- **Grid Data**: PBCT charts, K reading measurements
- **Mixed**: Slit lamp examination combines categorical findings with clinical descriptions

---

## Implementation Considerations

### Frontend Display
- Use collapsible sections for large forms
- Implement side-by-side OD/OS comparison views
- Table-based layouts for structured bilateral data
- Tab navigation between pages/major sections

### Data Extraction
- Segment codes should match section headers
- Preserve bilateral structure (OD/OS pairing)
- Support grid/table data structures (PBCT, IOP)
- Handle time-series data (diurnal IOP variations)
- Extract both categorical and free-text fields

### Medical Terminology
- **OD**: Oculus Dexter (Right Eye)
- **OS**: Oculus Sinister (Left Eye)
- **CDR**: Cup-to-Disc Ratio
- **IOP**: Intra Ocular Pressure
- **POAG**: Primary Open Angle Glaucoma
- **PERRLA**: Pupils Equal, Round, Reactive to Light and Accommodation
- **RAPD**: Relative Afferent Pupillary Defect
- **MGD**: Meibomian Gland Dysfunction
- **HVF**: Humphrey Visual Field
- **GHT**: Glaucoma Hemifield Test
- **VFI**: Visual Field Index
- **OCT**: Optical Coherence Tomography
- **OSDI**: Ocular Surface Disease Index
- **PBCT**: Prism Bar Cover Test
