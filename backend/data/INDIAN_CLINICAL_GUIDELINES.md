## Notes for Ingestion

### File Naming Convention
```
{organization}_{topic}_{year}.pdf
Example: ICMR_STG_Vol1_2019.pdf
         IAP_Fever_Management_2022.pdf
         NNF_Neonatal_Sepsis_2021.pdf
```

### Metadata to Extract
- `source_name`: Organization abbreviation (ICMR, IAP, NNF, etc.)
- `source_organization`: Full organization name
- `document_title`: Document title
- `specialty`: Mapped specialty (general_medicine, pediatrics, etc.)
- `topics`: Array of relevant topics
- `publication_year`: Year of publication
- `evidence_level`: If mentioned (Level A, B, C, Expert Consensus)

### Specialty Mapping
```python
GUIDELINE_SPECIALTY_MAP = {
    "ICMR_STG": "general_medicine",
    "IAP": "pediatrics",
    "NNF": "neonatology",
    "FOGSI": "obstetrics",
    "IPS": "psychiatry",
    "CSI": "cardiology",
    "AIOS": "ophthalmology",
    "RSSDI": "endocrinology",
    "API": "general_medicine",
    "SEMI": "emergency_medicine",
}
```
